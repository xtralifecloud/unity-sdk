Detailed Description

Constant string holder.

  • Automatically initializes to NULL
  • Duplicates the assigned string
  • Frees the assigned string upon destruction or assignment of a different one Miniature version of std::string. Equivalents (char* / cstring): { { char *str = NULL; str = strdup("hello"); cstring str = "hello"; printf("%s world", str); printf("%s world", str); if (str) delete str; } }
